# Gh0st
Auto Enumeration Script

- Install: </br>

           git clone https://github.com/getdrive/Gh0st
           
- Install dependences: </br>

          chmod +x install; bash install
           
- Run the script: </br>

          chmod + ghost.sh; bash ghost.sh
